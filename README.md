Pokedex construida com Python
Instale as seguintes bibliotecas para o código funcionar:
Pypokedex, Pillow e urllib3
pip install pypokedex 
pip install pillow 
pip install urllib3 

Essa Pokedex busca os pokemons pelo número ou nome.
